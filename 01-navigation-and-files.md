# Navigation and file system basics

## Goal
Move around the Linux file system confidently and always know where you are.

## Commands

### pwd
Show your current directory

pwd

---

### ls
List files and folders

ls
ls -l
ls -a
ls -la

---

### cd
Change directories

cd /etc
cd ..
cd ~
cd -

---

### tree
Show folder structure

tree
tree -L 2

## Notes
- ls -a shows hidden files that start with a dot  
- cd - returns to the previous directory
