# Linux Commands Reference

Practical Linux command notes for:

- IT support
- Red Hat labs
- Cybersecurity fundamentals

This repository is part of my hands-on learning journey while preparing for a career in Network Support and Cybersecurity.

---

## What this repo contains

- Navigation and file system basics  
- Viewing and editing files  
- Searching and text processing  
- Permissions and ownership  
- Users and groups  
- Processes and services  
- Networking essentials  
- Package management  
- Storage and file systems  
- Archives and compression  
- System information and logs  
- Shell basics and scripting  

Each section is written in beginner-friendly language and practiced inside a Linux virtual machine.

I am currently studying Cybersecurity and IT at Greenville Technical College and building real technical skills through:

1. Open a section file  
2. Run each command in a Linux terminal  
3. Observe the output  
4. Practice until the command feels natural
